# Sent

> *"As the Father has sent me, I am sending you."* — John 20:21

**Sent** is a "Duolingo for missionaries" — short, daily, gamified pre-field training so anyone (short-term
mission teams, church planters, individuals) can learn to share the Gospel in a specific language and
culture **before they go**. A project of **Favier's Lighthouse**.

Starter track: 🇫🇷 **France → French**.

---

## Repo layout

```
sent/
├── base44/                  ← the Base44 app definition (what Base44 reads / syncs)
│   ├── config.jsonc         ← app name "Sent" + build commands
│   └── entities/            ← the data model, one .jsonc per entity
│       ├── Country.jsonc  Language.jsonc  Phrase.jsonc  Verse.jsonc
│       ├── Lesson.jsonc   ChecklistItem.jsonc  TripPlan.jsonc
│       ├── Progress.jsonc  User.jsonc
├── standalone/              ← no-backend PWA: the runnable MVP demo (works TODAY, no credits)
│   ├── index.html  manifest.json  sw.js
├── docs/
│   ├── FRANCE_CONTENT_PACK.md     ← sourced briefing, phrases, verses, culture, checklist
│   └── DAILY_TRACK_LOGIC.md       ← how the countdown maps lessons across days
└── README.md
```

Once Base44 builds the UI, it will add the Vite/React app at the repo root
(`src/`, `package.json`, `vite.config.js`, `tailwind.config.js`, etc.) and keep `base44/` in sync —
the same structure as the `nlc-outreach` repo.

## Run the standalone demo

```bash
# quickest: just open it
#   open standalone/index.html in a browser

# proper (PWA + offline + installable on iPhone):
cd standalone
python -m http.server 8080     # or: npx serve .
#   then open http://localhost:8080
```

Audio is real: tap 🔊 on any phrase/verse to hear it in French (browser `SpeechSynthesis`, `fr-FR`).
Progress, XP and streak save to `localStorage`.

**Install on iPhone 16 Pro:** serve it, open `http://<computer-ip>:8080` in Safari, Share → Add to Home Screen.

## Base44 pickup (when credits regenerate)

The Base44 app already exists with all entities created and France content seeded
(app id `6a2818345fdbb4a6fd5206dd`). To finish it the NLC way:

1. In Base44, **build the Phase 1 UI** against the existing entities (onboarding/countdown, Today,
   Learn library, Nation briefing, Trip checklist, Me/globe/commission).
2. **Connect the Base44 app to this GitHub repo** (Push to GitHub) so it syncs `src/` + configs down —
   exactly like `nlc-outreach`.
3. From there, edit in Base44 or locally (`npm install && npm run dev`).

See `docs/` for the content and track logic the UI should render.
